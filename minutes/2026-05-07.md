### Libraries and Mints
* CDK
    * [cdk - v0.16.0](https://github.com/cashubtc/cdk/releases/tag/v0.16.0)
        * NUT-28 (Pay-to-Blinded-Key): Privacy-enhancing P2PK variant that blinds keys before spending
        * NUT-29 (Batch Minting): Support for batching mint quotes for improved efficiency.
        * BIP-321 Support: Complete parsing and creation of Bitcoin payment requests (BIP-321).
        * WalletTrait: Unified abstraction for all wallet types, simplifying integrations and FFI bindings.
        * Improved WASM Support: Refactored WebSocket client and time handling for better browser compatibility.
        * CLI Enhancements: BIP-353 resolution, NUT-26 PR support, and improved mint management commands.

    * [cashu-ts - v4.2.1](https://github.com/cashubtc/cashu-ts/releases/tag/v4.2.1)

    * [cashu-ts - v4.0.0](https://github.com/cashubtc/cashu-ts/releases/tag/v4.0.0)
        * Exact amount handling with Amount and bigint support across the API, eliminating silent precision loss (eg large millisat amounts).
        * Generic mint/melt APIs for custom payment methods, so apps can integrate new rails without waiting on library-specific support.
        * NUT-29 batch minting support for more efficient multi-quote mint workflows.
        * Better production resilience with explicit 429 handling, Retry-After parsing, and response metadata for rate-limit-aware clients.
        * Stronger security defaults via DLEQ verification hardening, optional strict signature enforcement, and tighter P2PK validation.
        * Improved privacy and compatibility through hardened request fingerprinting behavior and more flexible request-header overrides.
        * Slimmer ESM only package (ESM can finally be loaded from common.js on node)
    * [cashu-ts - v3.6.4](https://github.com/cashubtc/cashu-ts/releases/tag/v3.6.4) 
* coco
    * [coco - v1.0.0](https://github.com/cashubtc/coco/releases/tag/v1.0.0)
        * Stable storage-agnostic core for proof management, mint sync, quote lifecycle, counter tracking, wallet balances, operation recovery, and typed events.
        * Maintained adapters for Node SQLite, Bun SQLite, IndexedDB, and Expo SQLite.
        * React integration through operation-oriented hooks and provider state.
        * Plugin lifecycle hardening with idempotent initialization and duplicate-registration protection.
        * Structured wallet balance APIs via manager.wallet.balances.byMint() and manager.wallet.balances.total().
        * More complete receive and melt history tracking across prepared, pending, finalized, and rolled-back operation states.
        * Recovery support for executing melt operations through manager.ops.melt.refresh().
        * SQLite migration compatibility repairs and transaction isolation fixes across SQLite-backed adapters.
        * Expanded docs for send, receive, mint, and React operation lifecycles.
    
* Numo

    * [numo - v1.5](https://github.com/cashubtc/Numo/releases/tag/v1.5)
        * Rock Solid NFC: We've implemented major stability fixes for Tap2Pay. NFC transactions will no longer fail after retries or when resuming the app from the background, ensuring a smooth checkout experience every time.
        * Network Safety Trigger: Numo now intelligently checks for active internet connectivity. If you're offline, the app will prevent the generation of payment requests to save you and your customers from failed transactions.
        * Sales Summary: Gain better insights into your business! A brand new Sales Summary dashboard is now accessible directly from your transaction history.
        * Advanced History Filters: Easily navigate your past payments with a new expandable filter bar. You can now filter transactions by Settled or Pending statuses and apply Custom Date Ranges.
        * Unified Details & Receipts: We've merged the receipt screen directly into the transaction details, offering a much cleaner and more seamless experience when reviewing past payments.
        * Default Payment Methods: You can now configure your preferred default payment method in the app settings, allowing you to get paid exactly how you want, faster.
        * Reliable Exchange Rates via Yadio.io: We've switched our fiat exchange rate provider from Coinbase to Yadio.io. This change is especially important for LATAM currencies, providing more reliable data and preventing blockages in countries where Coinbase is blacklisted.
        * New Languages: Numo now fully supports Korean and Japanese translations! 🇰🇷🇯🇵
